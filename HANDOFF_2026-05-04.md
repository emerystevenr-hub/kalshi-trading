## Critical Operational Lessons

1. **Reconciler timezone gate:** use `target_date + 32h UTC` to handle PDT (latest US zone) + 1h NWS publish buffer. The naive `target_date >= today_utc` gate caused the Apr 27 partial-day settle bug three times before structural fix. **Don't relax this.**

2. **Kalshi market statuses:** Both `settled` AND `finalized` are terminal. The `result` field is source of truth. Daily polling markets and event markets use different statuses. T2 reconciler initial bug cost 27h of stuck positions. **Always check both.**

3. **NWS actuals merge-on-date:** running daily aggregates from Mesonet update through the day. Must merge max/min as new data lands, not skip-if-date-exists. Apr 27 partial-day wipeout taught this.

4. **σ-too-tight from small n:** caused pathological 0.0/1.0 probabilities. Defenses now in place: probability clip [0.05, 0.95], boundary filter (skip clipped strikes), EV filter (skip negative EV). **Don't remove these.**

5. **Side-aware calibration math:** `_model_win_prob` returns `our_p` for YES, `1-our_p` for NO. Mixing them wrecks the cal ratio (1.81 → 0.67 corrected on T1). **Use `_model_win_prob` everywhere, never raw `our_p`.**

6. **Real edge vs mid edge:** when paying yes_ask to enter, edge = our_p - yes_ask, not our_p - mid. The mid formula overstates YES edge by half-spread. T3c had this bug for 24h before catch on 2026-05-03.

7. **Daemon paths:** `Path.home()` resolves to the bash sandbox user inside the agent shell. Mac processes run from `/Users/stevenemery/Documents`. When the agent runs scripts via bash sandbox, files write to sandbox `~/Documents`, not Mac `~/Documents`. Use `HOME=/sessions/vigilant-hopeful-cannon/mnt python3 …` to redirect to Mac mount when needed.

8. **Picks pipeline: archetype-driven, not whitelist-driven.** The previous calendar_fade hardcode hid the fact that calendar_fade is currently 0/2. The new calibration-driven filter forces theses to earn their right to fire from settled outcomes, n≥3.
